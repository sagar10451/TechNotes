# AWS AI/ML Services

## Amazon Rekognition
- Detect objects, faces, age, emotions
- Count how many people in picture
- Celebrity recognition

## Amazon Transcribe
- Convert speech to text
- Removes PII (Personally Identifiable Information)
- Convert multiple languages at once

## Amazon Polly
- Text to speech
- Correct pronunciation
- Acronyms (AWS = Amazon Web Services)
- Supports SSML (comma, pause, etc.)

## Amazon Translate
- Language translation
- Website localization

## Amazon Lex + Connect

### Amazon Lex
- Interact and replies
- Powers Alexa devices

### Amazon Connect
- Call center solution
- Manages the call
- Routes the call

## Amazon Comprehend
- Finds insights and relationships in text
- Sentiment analysis

## Amazon Comprehend Medical
- Finds insights and relationships in clinical reports
- PHI (Protected Health Information) detection

## Amazon SageMaker
- For developers to build, manage, and deploy ML models

## Amazon Kendra
- Document search engine
- Ask questions and get answers from text

## Amazon Personalize
- Personalized recommendations
- Product recommendations
- Item ranking
- Customize marketing campaigns

## Amazon Textract
- Extract text and handwriting from scanned documents
